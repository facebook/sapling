# utils.ts

## Purpose

Shared utility functions used across ISL client and server. Provides common helpers for string manipulation, type checking, array operations, and data structures.

- `notEmpty()` - Type guard for filtering null/undefined
- `nullthrows()` - Assert non-null value
- `randomId()` - Generate random ID string
- `defer()` - Create externally-resolvable promise
- `basename()` - Extract filename from path
- `partition()` - Split array by predicate
- `group()` - Group array by key function
- And 20+ other utilities
